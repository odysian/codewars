# Strings

String manipulation and analysis solutions covering various text processing techniques.

## Subcategories

### string-filtering/
- **disemvowel-trolls.ts** - Character filtering based on conditions
- **reversed-string.ts** - String reversal operations

### string-manipulation/
- **descending-order.ts** - String to number conversion and sorting
- **string-incrementer.py** - String parsing and arithmetic operations

### string-analysis/
- **palindrome.ts** - Palindrome detection using two-pointer technique
- **detect-pangram.py** - Character set analysis for pangram detection

### string-encoding/
- **hashtag-generator.py** - String formatting and hashtag creation
- **vowel-code.py** - Character substitution and encoding
